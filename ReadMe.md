## Abstract

> 作为学习Django实践的项目，这里也用博客项目开始，该项目除了默认按时间排序的归档外
> 添加了按category类别排序和时间排序归档(`/blog/year/2017/`)
> 项目类别和about类别留待开发

---

## 项目环境

+ Python 3.5.3
+ Django 1.11.4
+ Bootstrap 3
